    Torch
    The aim of the project Torch is to make the visually impaired people's lives easier. The core of the project is an API that can be accessed over HTTPS. This API serves functionalities that are powered with machine learning techniques such as CNN's k-NN's etc. Some example functionalities are; banknote-detection, ocr, color-detection and object-detection. All of these services are accepting an image input. An example Django server that serves this API and an example PWA project was also developed. This project is soon to be published and repositories will be available to the public.

    LockerApp is a password manager. The application encrypts the passwords before saving them to an online database. It is developed with React-Native for the mobile and with NodeJS, jQuery for the web. It can be accessed at web or Google Play Store.
